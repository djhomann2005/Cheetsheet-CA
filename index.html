Adding Web Fonts

Adding web-based fonts to a website can be very simple. There are two primary ways to accomplish this. One is adding a link to the web font in the html file. The other is to download the font files locally and refer to them in the CSS sheet.

HTML Method

From the page of the web-based font you want to use, copy the link to the font. Paste this into the <head> section of the HTML document before the link to your CSS sheet (if you have one). This should be ready to use in your CSS sheet now.

CSS Method

For the CSS method to work, you must have the files stored somewhere with the website documents. You will want to know the relative path from the CSS document to the font files. These files should be .tff, .moff or .moff2 files (or a combination of these).
At the beginning of the CSS document, create a CSS ruleset with the selector @font-face. Within the declaration, name this font by using the property font-family: followed by what you want to call this font. Your name for the font should be enclosed in double quotes (""). Then, use the property src: with the value url(relative-path-here). Now you can use the font-family: property in other CSS rulesets with the value of the name you gave the font.

Code Function Code Syntax
HTML Link <link href="link_to_web_font">
CSS Selector @font-face
Font Name font-family
Font File Location url(relative-path)

Usage

Now that you have the associated with your website, you can use it in your CSS file. This is true whether you used the <link> in your HTML or the @font-face in your CSS. The variations of the font available are determined by the selections you made when choosing the font. You may have different font weights and styles, and you can change the size of the text. Use the table below for CSS declarations you can use to style your text. Note: some options may have no function if the font you used doesn't offer the options.

Font Change CSS Syntax Options
Weight font-weight 100-900 in hundreds
Size font-size # px/em/%
Style font-style normal, italic, bold
